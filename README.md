# Registration Form Validation

Форма регистрации на C# Windows Forms с валидацией полей через регулярные выражения.

<img width="380" height="405" alt="Снимок экрана 2026-03-02 153752" src="https://github.com/user-attachments/assets/3d201a16-bf45-4dc5-85ed-f497f43a4894" />

## Функциональность

- Поля: имя, логин, пароль, email
- Проверка через регулярные выражения
- Проверка уникальности email
- Сохранение в файл log.txt

## Интерфейс

- textBox1 — имя
- textBox2 — логин
- textBox3 — пароль
- textBox4 — email
- button1 — регистрация

## Куда можно встроить

- В приложения с регистрацией
- В системы аутентификации
- В учебные проекты

## Требования

- Windows
- .NET Framework / .NET Core с WinForms

## Примечание

Данные сохраняются в файл log.txt в папке приложения.
